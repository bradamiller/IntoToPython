# Slide Outline Template for Module 1 Lessons

> This file shows the structure and content of slide outlines for Module 1 lessons. See individual lesson slide outlines for complete content.

## General Slide Outline Structure

Each lesson has 10-12 slides organized as follows:

### Slide 1: Title & Agenda
- Lesson title
- Learning objectives (3-4 bullet points)
- Agenda / lesson flow

### Slides 2-3: Hook & Introduction
- Hook question or real-world connection
- Key vocabulary
- Why this matters

### Slides 4-7: Guided Learning
- Concept explanation with visuals
- Examples (Blockly or Python code)
- Worked problems
- Teacher demonstrates concept

### Slides 8-10: Practice & Application
- Student exercises
- Pair/group activities
- Interactive problem-solving

### Slide 11: Reflection & Assessment
- Exit ticket / formative assessment
- Connections to next lesson
- Summary of key ideas

### Slide 12 (Optional): Resources & Next Steps
- Links to code templates
- Homework / practice
- Preview of next lesson

---

## Notes for All Lessons

- **Use visuals**: Diagrams, code screenshots, robot photos
- **Live coding**: Show examples running on projector
- **Interactivity**: Ask questions, get predictions, celebrate ideas
- **Differentiation**: Each slide should have notes for advanced/struggling learners
- **Timing**: Plan for 45-50 minute classes; extend for 3-hour sessions
