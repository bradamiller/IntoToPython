When creating slide decks, use the sketch-style HTML format instead of the default PPTX generator style.

**Why:** User saw a YouTube video with hand-drawn/sketch-style slides and preferred that aesthetic — informal, illustration-heavy, whiteboard feel with hand-written fonts and SVG diagrams.

**How to apply:** Read `templates/sketch-style-html-slides.md` for full instructions. Reference implementation is `module-03-grid-driving/slides/01-introduction-to-the-grid-sketch.html`. Use Caveat/Patrick Hand/Fira Code fonts, teal/orange/green palette, inline SVG illustrations, and notebook-paper backgrounds. Generate `.html` files (not `.pptx`) with `-sketch` suffix.
