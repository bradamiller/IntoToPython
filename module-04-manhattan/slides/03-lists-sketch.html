What Is a List?

List: An ordered, mutable collection of values

Creating a list:
numbers = [1, 2, 3, 4, 5] names = ["Alice", "Bob", "Charlie"] empty = []
Key features:
Uses square brackets: [ ]
Values separated by commas
Can hold any type of value
Can grow and shrink (mutable!)
3 / 11

List Operations

Creating and accessing:
colors = ["red", "green", "blue"] print(colors[0]) # "red" print(colors[1]) # "green" print(len(colors)) # 3
Adding items with append():
colors = ["red", "green"] colors.append("blue") print(colors) # ["red", "green", "blue"] colors.append("yellow") print(colors) # ["red", "green", "blue", "yellow"]
append() always adds to the END of the list

Iterating Through a List

Use a for loop to visit each item:
colors = ["red", "green", "blue"] for color in colors: print(color)
red green blue
With index numbers:
for i in range(len(colors)): print("Item", i, "is", colors[i])
Item 0 is red Item 1 is green Item 2 is blue

Building a List Step by Step

Start empty, add items in a loop:

squares = [] for i in range(5): squares.append(i * i) print(squares)
[0, 1, 4, 9, 16]
This pattern is KEY for building paths!
6 / 11

A Path Is a List of Tuples

Combining lists and tuples:

path = [(0, 0), (1, 0), (2, 0), (2, 1), (2, 2), (2, 3)] print("Path has", len(path), "steps") print("Start:", path[0]) print("End:", path[-1])
Iterating through a path:
for step in path: row, col = step print("Visit:", step) print(" Row:", row, "Col:", col)

Building a Path with append()

Build a path programmatically:

path = [] # Add the starting position path.append((0, 0)) # Move down 2 rows path.append((1, 0)) path.append((2, 0)) # Move right 3 columns path.append((2, 1)) path.append((2, 2)) path.append((2, 3)) print("Path:", path)
Path: [(0, 0), (1, 0), (2, 0), (2, 1), (2, 2), (2, 3)]
Next lesson: We'll write code that CALCULATES the path automatically!

Common List Mistakes

Mistake 1: append() inside vs. outside loop
path = [] for i in range(3): position = (i, 0) path.append(position) # WRONG!

Only adds the last one!

path = [] for i in range(3): position = (i, 0) path.append(position) # Correct!

Adds each one inside the loop

Mistake 2: Confusing [ ] and ( )
path = [(0, 0), (1, 0)] # List of tuples ✓ path = ((0, 0), (1, 0)) # Tuple of tuples — not what we want
Mistake 3: Forgetting the list is empty
path = [] print(path[0]) # IndexError! Empty list has no items.
