Packaging Two Numbers Together

Problem: We need to store a position as TWO numbers (row and col)

Awkward approach

row = 2
col = 3
# Two separate variables
# for one position...
# 10 positions = 20 variables!

Better approach

position = (2, 3)
# One variable,
# one position — clean!

"How can we store two related numbers as a single value?"

What Is a Tuple?

A tuple is an ordered, immutable collection of values

Creating a tuple

position = (2, 3)
point = (0, 0)
destination = (3, 1)

Key features

Uses parentheses: ( )

Values separated by commas

Can hold any number of values

Values are ordered (first, second, third...)

Accessing Tuple Elements — Indexing

Use square brackets with an index number

position = (2, 3)

row = position[0]    # First element  → 2
col = position[1]    # Second element → 3

print("Row:", row)   # Row: 2
print("Col:", col)   # Col: 3
Critical Rule: Indexing starts at 0, not 1!
IndexValueMeaning
[0]2Row
[1]3Column

Unpacking & Creating Tuples

Tuple unpacking: get both values at once

Instead of using [0] and [1], unpack into named variables:

Indexing (verbose)

position = (2, 3)
row = position[0]
col = position[1]

Unpacking (cleaner)

position = (2, 3)
row, col = position

Creating a tuple from variables

Commas create tuples — parentheses are optional but recommended:

row = 2
col = 3

# Both create the same tuple
position = (row, col)
position = row, col
Unpacking gives readable names: row, col = position

Why Tuples Are Immutable

Immutable = cannot be changed after creation

This causes an error!

position = (2, 3)
position[0] = 5  # ERROR!

Create a new tuple instead

old_position = (2, 3)
new_position = (2, 4)  # New!

Why is this good for coordinates?

A coordinate IS a fixed point — (2, 3) is always row 2, col 3

If you need a new position, create a new tuple

Tuples for Robot Coordinates

Storing the robot's position

start = (0, 0)
destination = (2, 3)

# Unpack to get row and col
s_row, s_col = start
d_row, d_col = destination

print("Start:", start)
print("Going to:", destination)
print("Start row:", s_row)
print("Start col:", s_col)

Comparing coordinates

current = (1, 2)
target = (1, 2)
print(current == target)  # True!

other = (2, 1)
print(current == other)   # False!

Tuple Math — Manhattan Distance

How far apart are two coordinates?

start = (0, 0)
destination = (2, 3)

# Unpack both tuples
start_row, start_col = start
dest_row, dest_col = destination

# Calculate each distance
row_distance = dest_row - start_row
col_distance = dest_col - start_col

total = row_distance + col_distance
print("Total steps:", total)  # 5

Manhattan distance = row difference + column difference = total intersections to cross

Named after Manhattan's grid street layout

Common Tuple Mistakes

Mistake 1: Forgetting parentheses

Unclear

position = 2, 3  # Works, but not clear

Explicit

position = (2, 3)  # Better!

Mistake 2: Wrong index

Wrong

position = (2, 3)
row = position[1]  # Gets 3, not 2!

Correct

position = (2, 3)
row = position[0]  # Gets 2

Mistake 3: Trying to modify

TypeError!

position = (2, 3)
position[0] = 5  # Can't do this!

Make a new tuple

position = (2, 3)
position = (5, 3)  # New tuple!
