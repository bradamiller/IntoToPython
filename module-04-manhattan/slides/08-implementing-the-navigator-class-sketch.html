The __init__ Method

from XRPLib.differential_drive import DifferentialDrive class Navigator: def __init__(self, start, heading): self.position = start self.heading = heading self.drivetrain = DifferentialDrive.get_default_differential_drive()

Usage:

nav = Navigator((0, 0), "N") print("Position:", nav.position) # (0, 0) print("Heading:", nav.heading) # N

Important: The starting heading must match how you physically place the robot on the grid!

4 / 10

get_needed_direction()

Determine direction from current position to next position:

def get_needed_direction(self, next_pos): nxt_row, nxt_col = next_pos cur_row, cur_col = self.position row_diff = nxt_row - cur_row col_diff = nxt_col - cur_col if row_diff == 1: return "S" elif row_diff == -1: return "N" elif col_diff == 1: return "E" elif col_diff == -1: return "W"

Example:

nav = Navigator((0, 0), "N") direction = nav.get_needed_direction((1, 0)) print(direction) # "S" — need to go south (row increases)
5 / 10

turn_to()

Turn the robot to face the desired direction:

def turn_to(self, direction): if self.heading == direction: return # Already facing right way right_turns = { "N": "E", "E": "S", "S": "W", "W": "N" } left_turns = { "N": "W", "W": "S", "S": "E", "E": "N" } if right_turns[self.heading] == direction: self.drivetrain.turn(90) self.heading = direction elif left_turns[self.heading] == direction: self.drivetrain.turn(-90) self.heading = direction else: # 180 degree turn self.drivetrain.turn(180) self.heading = direction
6 / 10

drive_path()

The main method — drive through every step in the path:

def drive_path(self, path): for next_pos in path: if next_pos == self.position: continue # Skip the starting position # 1. Figure out which way to face direction = self.get_needed_direction(next_pos) # 2. Turn if needed self.turn_to(direction) # 3. Drive forward one intersection self.drivetrain.straight(30, 0.5) # 4. Update position self.position = next_pos print("Arrived at", self.position)
The loop: For each coordinate in the path, turn to face it, drive forward, update position.

Note: straight(30, 0.5) assumes 30 cm between intersections — adjust for your grid!
