What Is a Coordinate System?

A way to describe any position using numbers

Examples You Already Know

Battleship: "B4" = column B, row 4

Chess: "e5" = column e, row 5

Maps: latitude and longitude

Spreadsheets: cell A1, B3, etc.

Two numbers uniquely identify
any position on a 2D surface

3 / 10

Our Grid Convention — (row, col)

The XRP grid uses (row, column)

Convention

First number = row (how far down)

Second number = column (how far right)

Start at (0, 0) in the top-left corner

Row first, Column second!
(row, col)
4 / 10

Reading Coordinates

Practice: What are the coordinates?

A — Top-left corner: (0, 0)

B — One row down, two columns right: (1, 2)

C — Bottom-right of 4×4 grid: (3, 3)

D — Third row, first column: (2, 0)

Common Mistake!

(2, 3)

row 2, col 3

≠

(3, 2)

row 3, col 2

These are DIFFERENT positions!

The Physical Grid

Your robot drives on a physical grid

Black tape lines form a grid on the floor

Each intersection is a coordinate

The robot uses line-following to travel between intersections

How to Count

Start at (0, 0) — the home position

Moving forward (down) increases the row

Moving right increases the column

Black tape = roads
Intersections = coordinates
6 / 10

Why Coordinates Matter

Without Coordinates

go forward turn right go forward go forward turn left go forward...

Hard to plan, hard to change destination

With Coordinates

go from (0, 0) to (2, 3)

The robot can CALCULATE the path

Change destination? Just change the numbers!
