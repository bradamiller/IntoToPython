The Program Structure

The complete square program

board.wait_for_button()
print("Starting square pattern!")

for leg in range(4):
    print("Side", leg + 1, "of 4")
    drive_intersections(tracker, 2)
    tracker.turn_right()

print("Square complete!")

Each iteration:

1 Drives 2 intersections (one side)
2 Turns right (one corner)
3 Repeats for all 4 sides

Debugging Tips

Common problems and fixes

Robot doesn't return to start?

Check turns are exactly 90°

Count the intersections — each side should be exactly 2

Robot skips an intersection?

Slow down the drive effort

Check the clearing distance (8 cm) in drive_intersections

Robot veers after turning?

Make sure the robot is centered on the line before driving

Check that straight(8, 0.5) clears the intersection fully

General strategy: Change ONE thing, test, observe.
Never change two things at once!
